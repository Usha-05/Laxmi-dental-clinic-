# Laxmi Dental Clinic Website

Modern marketing site for Laxmi Face and Multispeciality Dental Hospital, built with Next.js and Tailwind CSS.

## Key Features
- Appointment booking modal that forwards submissions to email and WhatsApp
- Contact form with automatic WhatsApp redirection
- Treatment landing pages grouped by specialty
- Responsive layout with custom hero, testimonials, and gallery sections

## Getting Started
```bash
# install dependencies
npm install

# run locally
npm run dev

# build for production
npm run build
npm run start
```

Create a `.env.local` file for email and WhatsApp integrations. Refer to `SETUP_INSTRUCTIONS.md` in the repository for configuration details.

## Deployment
The app is optimized for deployment on Vercel or any Node.js hosting platform that supports Next.js.
